# Hangman
A simple [hangman](https://en.wikipedia.org/wiki/Hangman_(game)) game made in Julia.

Features:
- Input validation
- Hides word in console
- Renders man that is getting hung in Gtk
- Replay prompt built in
- Comments in code
- Screenshots included
